Machine learning is a vast subject and there is a lot to learn. Luckily, there are several books out there that can help us along the way. Below I list what I believe are the top 5 machine learning books that are currently out there.

<iframe src="//www.youtube.com/embed/D1J0RTsJuFA?wmode=opaque&amp;enablejsapi=1" height="480" width="854" scrolling="no" frameborder="0" allowfullscreen></iframe>

1. [Hands on Machine Learing](https://amzn.to/2HKBpgR) - [Second edition](https://amzn.to/2UehZqy) comes out in August 2019
2. [Machine Learning with Python Cookbook](https://amzn.to/2usyGAh)
3. [Python Data Science Handbook](https://amzn.to/2usyGAh) and for more pandas [Python for Data Analysis](https://amzn.to/2Oq1UZw) is also great
4. [Deep Learning with Python](https://amzn.to/2U2fOHg)
5. [Introduction to Statistical Elements](https://amzn.to/2FFIAor)
